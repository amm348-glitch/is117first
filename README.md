# My First Website

This is my first website created using HTML, CSS, and JavaScript. It is hosted on GitHub Pages.

## Files
- `index.html`: The main HTML file.
- `style.css`: The styling for the webpage.
- `scripts.js`: The JavaScript code for interactions.

## How to Use
1. Clone the repository or download the files.
2. Open `index.html` in a browser to view the website.
